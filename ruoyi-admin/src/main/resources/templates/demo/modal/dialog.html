設置選項

可以透過數據綁定或者Javascript來實現模態窗口的相關功能，如果使用數據綁定，可以為元素添加data-，如data-backdrop=""。

名稱 類型 預設值 說明
backdrop boolean 或 string 'static' true 遮罩層，或使用'static'指定遮罩層與關閉模態窗口不關聯
keyboard boolean true 按Esc鍵時退出模態窗口
show boolean true 初始化完成後顯示模態窗口
remote path false

推薦使用數據綁定方式，或使用 jQuery.load

遠程URL範例：

<a data-toggle="modal" href="remote.html" data-target="#modal">Click me</a>
